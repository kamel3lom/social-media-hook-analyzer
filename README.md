# Social Media Hook Analyzer

أداة ويب عربية/إنجليزية تعمل مباشرة من المتصفح لتحليل قوة الهوك في منشورات السوشيال ميديا أو عناوين الفيديوهات، ثم اقتراح نسخ أقوى مناسبة للمنصات التالية:

- YouTube
- TikTok
- Instagram
- X / Twitter
- Facebook
- LinkedIn

الأداة مصممة لتناسب الباحثين، صناع المحتوى، مهندسي البرومبت، أصحاب الخدمات، والمدربين.

## المزايا

- تحليل قوة الهوك من 100.
- تقييم تفصيلي لستة مؤشرات:
  - الوضوح
  - الفائدة
  - الفضول
  - التحديد
  - الاختصار
  - ملاءمة المنصة
- اقتراح نسخ أقوى للعنوان أو فكرة المنشور.
- دعم العربية والإنجليزية.
- دعم المنصات المختلفة بقواعد مختلفة.
- تقليل العناوين المضللة والـ Clickbait.
- سجل محلي لآخر التحليلات عبر `localStorage`.
- تصدير الاقتراحات بصيغة JSON وCSV.
- نسخ التقرير أو الاقتراحات مباشرة.
- يعمل دون خادم ودون قاعدة بيانات.
- قابل للنشر على GitHub Pages.
- قابل للتثبيت كتطبيق PWA.

## طريقة التشغيل محليًا

لا تحتاج إلى تثبيت حزم خارجية. افتح الملف:

```text
index.html
```

أو شغّل خادمًا محليًا بسيطًا:

```bash
python -m http.server 8000
```

ثم افتح:

```text
http://localhost:8000
```

## النشر على GitHub Pages

1. أنشئ مستودعًا جديدًا على GitHub.
2. ارفع الملفات كما هي إلى المستودع.
3. ادخل إلى:
   `Settings > Pages`
4. اختر:
   `Deploy from a branch`
5. اختر الفرع:
   `main`
6. اختر المجلد:
   `/root`
7. اضغط Save.

بعد لحظات سيظهر رابط الأداة على GitHub Pages.

## ملاحظات مهمة حول الدقة

هذه الأداة تستخدم خوارزمية تحليل معيارية Rule-Based وليست نموذج ذكاء اصطناعي متصلًا بالخادم. لذلك فهي آمنة وسريعة وتعمل دون إنترنت بعد التحميل، لكنها لا تدّعي أن تقييمها حكم نهائي على الانتشار. الهدف منها إعطاء مؤشر عملي يساعد في تحسين الصياغة قبل النشر.
